# beneficial-ownership-data
Expert Claude skill for beneficial ownership data, Beneficial Ownership Data Standard, international policy context, and country-specific registers.

Expert reference for beneficial ownership data, policy and the Beneficial Ownership Data Standard (BODS). 

Trigger whenever the topic touches on: BODS JSON (parsing, statement types, converting to SQLite/CSV/Parquet), Open Ownership tools (bodsdata, bods-dagre, bodsanalysis, CoVE-BODS, RDF vocabulary, data generator), UBO/beneficial ownership registries, FATF Recommendations 24 and 25, EU AML Regulation (AMLR 2024/1624 or AMLD6), BO register design or policy, country-specific BO data models (UK PSC, Norway, France, Indonesia, Denmark) or international BODS endorsements.

If the question is about who owns what — the data, tools, or policy around that — use this skill.
