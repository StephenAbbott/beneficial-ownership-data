# Claude skill to understand beneficial ownership data
Expert [Claude skill](https://code.claude.com/docs/en/skills) for beneficial ownership data, the [Beneficial Ownership Data Standard](https://standard.openownership.org/en/0.4.0/) (BODS), international policy context, and country-specific registers.

Download [latest version of the beneficial-ownership-data Claude skill](https://github.com/StephenAbbott/beneficial-ownership-data/releases/download/v1.7.0/SKILL.md).

Trigger whenever the topic touches on: BODS JSON (parsing, statement types, converting to SQLite/CSV/Parquet), Open Ownership tools (bodsdata, bods-dagre, bodsanalysis, CoVE-BODS, RDF vocabulary, data generator), UBO/beneficial ownership registries, FATF Recommendations 24 and 25, EU AML Regulation (AMLR 2024/1624 or AMLD6), BO register design or policy, country-specific BO data models (UK PSC, Norway, France, Indonesia, Denmark), international BODS endorsements or sectoral approaches to beneficial ownership transparency in procurement, extractives and fisheries.

If the question is about who owns what — the data, tools, or policy around that — use this skill.
